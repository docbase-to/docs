---
title: "Event Reference"
---

This page lists all the webhook events that Docbase can send to your application.

Every payload follows the base structure defined in the past, but the `data` object changes depending on the event.

## File Cabinet

<Tabs>
  <Tab title="file_cabinet.created">
    Triggered immediately after a file cabinet is succesfully created.

    ```json
    {
      "eventId": "4e556a05-f873-469f-ac89-9ff35584009b",
      "eventType": "file_cabinet.created",
      "subscriptionId": "fd111aac-697f-45e5-9dfe-1a7d9fb53d09",
      "timestamp": "2026-03-12T16:36:05.4036873Z",
      "data": {
        "id": "9168ea64-95a8-4cde-aa31-fa76442c06d2",
        "name": "Some name",
        "description": "Some description",
        "organizationId": "8b2ba8c4-11ac-45f6-a187-8037492ac30e",
        "createdAt": "2026-03-12T16:36:05.3283323Z"
    }
    ```
  </Tab>
  <Tab title="file_cabinet.updated">
    Triggered immediately after a file cabinet is succesfully updated.

    ```json
    {
      "eventId": "ef27413e-e603-4a9c-86cc-2508c88e31f2",
      "eventType": "file_cabinet.updated",
      "subscriptionId": "15270030-6f3f-4987-8355-7e00e1ead748",
      "timestamp": "2026-03-12T18:22:09.8717925Z",
      "data": {
        "id": "b932c9ca-21ba-4a47-87dd-052bcaf9805f",
        "name": "New name",
        "description": "Test description",
        "organizationId": "8b2ba8c4-11ac-45f6-a187-8037492ac30e",
        "createdAt": "2026-03-12T17:42:54.162972Z",
        "updatedAt": "2026-03-12T18:22:09.8442645Z"
    }
    ```
  </Tab>
  <Tab title="file_cabinet.deleted">
    Triggered immediately after a file cabinet is succesfully deleted.

    ```json

      "eventType": "file_cabinet.deleted",
    ```

  </Tab>
</Tabs>
